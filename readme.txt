== Description ==

Custom Content by (Shortcode & Elementor Extension) plugin Extend the Elementor Builder with ES Modules (ES Custom Content Elementor) display custom contents using shortcode or Elementor Extension.


= Features =

* Display contents by Shortcode `[es_content_elementor id="1"]`
* Display contents by Elementor Extension
* Add custom class by Elementor Extension


== Installation ==

1. Install automatically through the `Plugins`, `Add New` menu in WordPress, or upload the `custom-content-elementor` folder to the `/wp-content/plugins/` directory. 

2. Activate the plugin through the `Plugins` menu in WordPress. On the Admin screen use the Plugins side menu to select Custom Content Elementor to configure.  

3. You must add this shortcode `[es_content_elementor id="1"]` in a Page, Post, or Text Widget. You can find this shortcode on the cuntom contents page. Here is how to add the shortcode: Log into your blog admin dashboard. Click `Pages`, click `Add New`, add a title to your page, enter the shortcode `[es_content_elementor id="1"]` in the page. Uncheck `Allow Comments`, then click `Publish`. and also can use Elementor Extension for display contents. 

4. Updates are automatic. Click on "Upgrade Automatically" if prompted from the admin menu. If you ever have to manually upgrade, simply deactivate, uninstall, and repeat the installation steps with the new version.
